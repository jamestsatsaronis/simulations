pKa reference & method

An acidic group (loses H⁺) carries −1 when pH > pKa and 0 when pH < pKa. A basic group (gains H⁺) carries +1 when pH < pKa and 0 when pH > pKa. At pH = pKa the group sits at its midpoint: [HA] = [A⁻], so at any instant it is ionised in half the molecules and neutral in the other half. Charge is quantised — a single group is always either fully charged or fully neutral, never literally fractional — so the ±½ we use is the average charge across the population (equivalently, the time-average for one molecule that exchanges its proton rapidly). Net charge is the sum across every ionisable group; in a peptide only the two termini and the side chains ionise.

With the half-charge option on, some pH values are set exactly to a pKa, where that group is ionised in half the molecules, so on average it contributes ±½; every other group is still kept at least 1 unit from its pKa, so only that one group is fractional. With the option off, all pH values stay at least 1 unit from every pKa and answers are whole numbers. The exact line uses the Henderson–Hasselbalch equation. pKa values are teaching values close to Lehninger; terminal α-groups use a single generic value for consistency.
