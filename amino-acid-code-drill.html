Amino acid code drill

A self-paced flashcard drill for converting between the full name, the three-letter code and the one-letter code of the 20 standard amino acids. You are shown one form; supply the other two. Every round shows the full answer and the amino acid's category, so a miss is still a learning moment. Progress is not saved — reloading the page starts a fresh session.
